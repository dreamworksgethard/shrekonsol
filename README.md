# Shrek Symbol ($SHREK) — Meme Website

Simple, fun, clean one-page site for **Shrek Symbol $SHREK**.

## Run locally

From this folder:

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

## Update your info

Edit `script.js`:

- `CONFIG.contractAddress`
- `CONFIG.links.dex` / `cex` / `chart`
- `CONFIG.links.x` / `tg`
- `CONFIG.supply`, taxes, etc.
